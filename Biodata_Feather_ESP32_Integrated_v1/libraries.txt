LIBRERÍAS NECESARIAS PARA BIODATAFEATHER INTEGRATED v1.0
================================================================

INSTALAR DESDE EL GESTOR DE LIBRERÍAS DE ARDUINO:

1. WiFi (incluida con ESP32)
2. Wire (incluida con ESP32)
3. EEPROM (incluida con ESP32)

DESDE EL GESTOR DE LIBRERÍAS:

4. PubSubClient by Nick O'Leary
   - Versión: 2.8.0 o superior
   - URL: https://github.com/knolleary/pubsubclient

5. ArduinoJson by Benoit Blanchon
   - Versión: 6.21.0 o superior
   - URL: https://github.com/bblanchon/ArduinoJson

6. Adafruit TinyUSB Library
   - Versión: 2.2.0 o superior
   - URL: https://github.com/adafruit/Adafruit_TinyUSB_Arduino

DESDE GITHUB (si no están en el gestor):

7. Adafruit LTR329 LTR303 Library
   - URL: https://github.com/adafruit/Adafruit_LTR329_LTR303
   - Descargar ZIP e instalar desde Arduino IDE

8. DFRobot BME68x Library
   - URL: https://github.com/DFRobot/DFRobot_BME68x
   - Descargar ZIP e instalar desde Arduino IDE

CONFIGURACIÓN ADICIONAL:

- ESP32 Board Package (versión 2.0.0 o superior)
- Adafruit ESP32-S3 Feather Board Package

NOTAS:
- Asegúrate de tener la última versión del Arduino IDE
- Si tienes problemas con las librerías, prueba con versiones específicas
- Para ESP32-S3, asegúrate de tener el board package correcto instalado
